Indonesia Shipping Intelligence

인도네시아 해사청(kapal.dephub.go.id)의 선박 등록 데이터와 Inaportnet (monitoring-inaportnet.dephub.go.id)의 LK3 물동량 데이터를 매월 자동 수집하여 변경 사항을 추적합니다.

데이터 출처

  • 선복량: 56개 선종 코드 × 전체 페이지
  • 물동량: 모든 항구 × 직전 24개월 × 국내(dn)/국제(ln)

Pipeline

Python (requests + pandas + SQLAlchemy) → SQLite → 정적 JSON 추출 → GitHub Pages.
