📚 Glossary & Methodology

Indonesia Shipping Intelligence는 공개 데이터를 가공한 사실 기반 정보 레퍼런스입니다. 모든 수치는 공개 출처에서 자동 수집된 raw 데이터의 집계 결과이며, 가치 평가나 판단 의견을 포함하지 않습니다.

1. 데이터 출처 · 수집 주기

소스 URL 수집 주기 파일
Inaportnet LK3 monitoring-inaportnet.dephub.go.id 월 1회 (전월 마감 후) cargo_snapshot
선박 등록 (kapal) kapal.dephub.go.id 월 1회 vessels_snapshot / vessels_current
IDX 공시 idx.co.id 분기 1회 (FY 보고서 공시 시) data/companies_financials.yml

데이터는 SQLite (data/shipping_bi.db)에 저장되고, backend/build_static.py + scripts/build_derived.py가 정적 JSON으로 추출합니다.

2. 정의 · 분류 기준

  • Vessel Class — 5-tier sector + 11 vessel_class. 분류는 backend/taxonomy.py의 keyword rule table.
  • Tanker Subclass (6종) — Crude Oil · Product · Chemical · LPG · LNG · FAME / Vegetable Oil. JenisDetailKet (vessel) 또는 JENIS KAPAL (LK3) 라벨에 키워드 룰 적용.
  • Bongkar (B / dn) — 양하 (discharge), 항구 도착 시 내려진 화물. Muat (M / ln) — 선적 (load), 항구 출발 시 적재된 화물. 표기에 따라 dn(국내)/ln(국제) 분리 컬럼도 사용.
  • HHI (Herfindahl-Hirschman Index) — 운영사별 GT 점유율의 제곱합 × 10,000. 0~10,000 스케일. 1,500 미만 = 분산, 1,500–2,500 = 중간, 2,500 이상 = 집중 (KPPU 가이드 참조).
  • CAGR (24M) — (ton_last_12m / ton_prev_12m) ^ (1/2) - 1. prev/last 윈도우가 둘 다 정확히 12개월일 때만 산출, 그 외엔 null.
  • OD pair — origin port → destination port 항로. tanker_flow_map.lanes 기준 24M 집계.

3. 한계 · 주의사항

  • Owner 명 모호성 — 동일 회사가 LK3와 vessel 등록에서 다른 표기 (PT./PT/Tbk 접미사 유무, 띄어쓰기 차이) 사용. 정규화는 _norm_company (대문자, 공백·구두점 정리)로 best-effort 매칭하지만 100% 보장 불가.
  • LK3 신고 기반 — 데이터는 항구 운영자가 Inaportnet에 신고한 값 그대로. 신고 누락·중복·재신고가 발생할 수 있으며 사후 정정도 수개월 후 반영됨.
  • partial 월 — 스크래핑 시점의 당월 데이터는 대부분 부분 데이터. kpi_summary.latest_period_is_partial_data_dropped 플래그로 표시되며, derived 빌드는 partial 월을 제외해 latest_lk3_month를 한 달 뒤로 잡음.
  • Top-50 절단 — tanker_focus.fleet_owners가 GT 상위 50개로 truncate되어, 운영사 수 / HHI 등 운영사 기반 집계는 long-tail을 반영하지 못함.
  • baseline snapshot — 현 시점 단일 snapshot (2026-05) 만 존재. MoM 변동 기반 이벤트 (port_change ±50% / fleet_change ±3 / volume_change ±30%)는 두 번째 snapshot 도착 시 자동 활성화.

본 사이트는 공개 데이터(kapal.dephub.go.id · monitoring-inaportnet.dephub.go.id · IDX 공시)를 가공한 정보 레퍼런스이며, 투자 권유나 자문이 아닙니다.

데이터의 정확성·최신성은 보장되지 않으며, 모든 분류 · 집계 · 자동 텍스트는 사실 진술까지만 제공합니다. 특정 회사 · 티커 · 항구를 평가하거나 권장하지 않으며, 가치 판단 어휘를 사용하지 않습니다.
