💼 Listed Operators

IDX 상장 해운 운영사의 재무 + 선대 비교. 단위: IDR billion (10억 루피아) · 선대 GT는 kGT.

정렬은 표 헤더 클릭

매출 × 순이익률 (선택 연도)

x = 매출 (log), y = 순이익률 (%), 크기 = 선대 GT. 영역 라벨 없음.

기업별 재무 비교
