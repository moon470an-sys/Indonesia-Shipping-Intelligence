Tanker Sector — Regulatory & Technical Notes

본 문서는 인도네시아 해운(특히 탱커) 섹터에 적용되는 규제·기술 표준의 요약 정보입니다. 법률 자문이 아니며, 실제 운항·계약 의사결정은 전문 법무·해운 컨설팅을 거쳐야 합니다.

1. 카보타지 원칙 (UU No. 17/2008 Pelayaran)

  • 근거: Undang-Undang Republik Indonesia Nomor 17 Tahun 2008 tentang Pelayaran (해운법), 제8조.
  • 요지: 인도네시아 영해에서의 국내 해상운송은 인도네시아 국적 선박, 인도네시아 선원, 인도네시아 해운회사가 운영해야 한다.
  • 적용: 국내 (dn) 항행 100%. 국제 (ln) 운항은 카보타지 적용 외.
  • 참조: (BPK)

2. SIUPAL (Surat Izin Usaha Perusahaan Angkutan Laut)

  • 근거: PP No. 20/2010, PM Perhubungan No. 93/2013 (개정 PM No. 28/2024).
  • 요지: 인도네시아에서 해상운송 사업을 영위하기 위한 면허. 자기 소유 선박 1척 이상(국적 등록) 필요.
  • 취득 절차: NIB (OSS-RBA) → SIUPAL 신청 → DJPL (Direktorat Jenderal Perhubungan Laut) 심사·발급.
  • 참조: · DJPL 공식 가이드.

3. IBC Code — Type 1 / 2 / 3 (Chemical Tanker)

  • 근거: IMO International Code for the Construction and Equipment of Ships Carrying Dangerous Chemicals in Bulk (IBC Code), Resolution MSC.4(48), 1983 (개정 다수).
  • 분류: 화물의 위험도에 따라 선체 구조 요구가 결정됨.

- Type 1: 가장 엄격. 충돌·좌초 시 누출 방지 위해 완전 이중 선체 + 화물 탱크가 선측에서 충분히 떨어져야 함. 매우 위험한 화물 (예: Acrylonitrile, Carbon Disulphide). - Type 2: 중간 수준. 이중 선체 일반적, 일부 화물 탱크 위치 요구. 대부분의 산성·알칼리 화물. - Type 3: 가장 완화된 요건. 단일 선체 가능한 경우 있음. 비교적 위험도 낮은 화물 (예: Vegetable Oil — 일부 등급은 IBC 외).

  • 참조: IMO MSC.4(48), MSC.176(79) 개정.

4. IBC 2G vs IGC 2G

  • 공통: 두 코드 모두 Type 2 / Class G 등급 표기에 "2G"를 쓴다는 점에서 혼동 주의.
  • IBC 2G: 화학 액체 (chemical liquid) 운반선 — Type 2 등급 + Type G 화물 탱크 (independent gravity tank).
  • IGC 2G: 액화 가스 (liquefied gas — LPG/LNG) 운반선 — IGC Code (International Code for the Construction and Equipment of Ships Carrying Liquefied Gases in Bulk, Resolution MSC.5(48))의 Type 2G 등급. 산화제 성격이 강한 가스 화물 (Ammonia, Vinyl Chloride 등).
  • 요점: 같은 "2G" 라벨이라도 적용 코드가 다르므로, 화물 종류(액체 화학 vs 액화 가스)부터 확인.

5. MARPOL Annex I vs Annex II

  • 근거: International Convention for the Prevention of Pollution from Ships (MARPOL 73/78), IMO.
  • Annex I — Oil: 원유, 석유제품(휘발유·디젤·연료유 등) 운송에서 발생하는 유성 잔사·평형수의 배출·기록 규정.

- 적용: Crude Oil tanker, Product tanker, Combination Carrier (OBO/OBO-OO) 등. - 핵심 의무: ODMCS (Oil Discharge Monitoring & Control System), Oil Record Book (ORB), SOPEP.

  • Annex II — Noxious Liquid Substances (NLS): 액체 화학 화물 (Category X / Y / Z / OS 4단계 분류) 의 배출·취급·세척수 처리.

- 적용: Chemical tanker, IBC Code 적용 선박. - 핵심 의무: Cargo Record Book, Procedures and Arrangements (P&A) Manual, Prewash 절차.

  • 요점: 단일 선박이 두 Annex 모두 적용받을 수 있음 (예: oil/chemical tanker).

6. PT PMA (Penanaman Modal Asing) 일반 절차

  • 근거: UU No. 25/2007 Penanaman Modal, PP No. 20/2010, OSS-RBA 시스템 (BKPM Reg. No. 4/2021 → 7/2025).
  • 요지: 외국인 지분이 포함된 인도네시아 법인 (PT PMA) 설립을 통한 해운업 진출.
  • 단계:

1. 법인 정관 작성 (Notaris) + Akta Pendirian. 2. SK Kemenkumham (법무인권부 승인). 3. NIB (OSS-RBA) 발급 → 사업분야 KBLI 코드에 따라 자동 분류. 4. 해운업 KBLI (예: 50111 Domestic sea passenger transport, 50114 Domestic sea freight transport) — 외국인 지분 한도는 Negative Investment List (DNI) → 현재 Positive Investment List (DPI, Perpres 49/2021) 확인 필요. 5. SIUPAL 등 분야별 라이선스 추가 취득.
